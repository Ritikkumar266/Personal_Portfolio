# Ritik Kumar - Portfolio

A modern, responsive portfolio website built with Next.js 16, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- ✨ Modern and clean design with smooth animations
- 🎨 Gradient backgrounds and hover effects
- 📱 Fully responsive across all devices
- 🚀 Fast performance with Next.js 16
- 🎭 Framer Motion animations
- 🎯 Smooth scroll navigation
- 📧 Contact form
- 🌐 Social media links

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

### Update Personal Information

1. **Hero Section** (`components/Hero.tsx`):
   - Update your name, title, and description
   - Change social media links (GitHub, LinkedIn, Twitter)

2. **About Section** (`components/About.tsx`):
   - Modify the about text and features

3. **Skills Section** (`components/Skills.tsx`):
   - Add or remove skills and technologies
   - Update skill categories

4. **Projects Section** (`components/Projects.tsx`):
   - Add your projects with descriptions
   - Update GitHub and demo links
   - Change project tags

5. **Contact Section** (`components/Contact.tsx`):
   - Update email, phone, and location
   - Configure form submission logic

6. **Metadata** (`app/layout.tsx`):
   - Update SEO metadata, title, and description

## Build for Production

```bash
npm run build
npm start
```
